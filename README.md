Yapay Zeka Destekli Tanı ve Semptom Analiz Sistemi

Bu proje, kullanıcıların girdikleri semptomlara dayanarak olası hastalıkları tahmin eden ve semptomlar arasındaki gizli ilişkileri veri madenciliği teknikleriyle analiz eden bir web uygulamasıdır.

Öne Çıkan Özellikler
* Hastalık Tahmini: Decision Tree algoritması kullanılarak eğitilen model ile yüksek doğrulukta tanı simülasyonu.
* Birliktelik Kuralları: Apriori algoritması kullanılarak hangi semptomların birlikte görülme eğiliminde olduğunun analizi.
* Dinamik Ağ Grafiği: Semptomlar arasındaki ilişkilerin pyvis kütüphanesi ile interaktif ve görsel olarak sunulması.
* Kullanıcı Dostu Arayüz: Streamlit ile geliştirilmiş modern ve hızlı web arayüzü.

Kullanılan Teknolojiler
* Python
* Decision Tree
* Mlxtend (Apriori - Association Rules)
* Streamlit
* Pyvis, Pandas
